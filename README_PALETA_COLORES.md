# Paleta de Colores - Marcos Morici Tractores

## Descripción
Esta paleta de colores está basada en los logos oficiales de Marcos Morici Tractores, creando una identidad visual coherente y profesional para la página web especializada en la venta de equipos, piezas y repuestos de tractores.

## Paleta Principal

### Color Primario - Azul Oscuro (Navy Blue)
- **Hex:** `#1E3A5F` o `#1A2F4F`
- **RGB:** `rgb(30, 58, 95)` o `rgb(26, 47, 79)`
- **HSL:** `hsl(214, 52%, 25%)` o `hsl(214, 50%, 20%)`
- **Uso:** 
  - Header principal
  - Logo y branding
  - Texto principal y títulos
  - Botones de acción primarios
  - Footer (fondo)
  - Bordes y elementos estructurales
  - Navegación principal

### Color Secundario - Verde (Montañas)
- **Hex:** `#4A7C59` (verde medio) / `#6B8E5A` (verde claro) / `#3D5F47` (verde oscuro)
- **RGB:** `rgb(74, 124, 89)` / `rgb(107, 142, 90)` / `rgb(61, 95, 71)`
- **HSL:** `hsl(140, 25%, 39%)` / `hsl(100, 22%, 45%)` / `hsl(140, 22%, 31%)`
- **Uso:**
  - Acentos y elementos destacados
  - Botones de acción secundarios
  - Etiquetas de productos (tags)
  - Sección de newsletter (alternativa)
  - Hover states
  - Elementos decorativos

## Tipografía

### Fuentes Recomendadas
Basadas en el estilo del logo (sans-serif, bold, uppercase):

**Opción 1 - Fuentes Modernas y Versátiles:**
- **Principal:** 'Inter', 'Roboto', 'Open Sans', 'Montserrat'
- **Características:** Sans-serif, buena legibilidad, múltiples pesos

**Opción 2 - Fuentes Industriales (más acorde al logo):**
- **Títulos:** 'Bebas Neue', 'Oswald', 'Anton' (para títulos y nombres)
- **Cuerpo:** 'Roboto', 'Open Sans', 'Lato' (para texto descriptivo)

**Características del Logo:**
- Sans-serif
- Bold/Heavy para títulos principales
- Uppercase para nombres de marca y títulos importantes
- Letras angulares y fuertes (similar a la "M" del logo)

### Aplicación de Colores en Tipografía
- **Títulos Principales:** Azul oscuro (`#1E3A5F`), bold, uppercase
- **Títulos Secundarios:** Azul oscuro o gris azulado (`#5A6C7D`), semi-bold
- **Texto Normal:** Azul oscuro o gris azulado, regular
- **Texto sobre Fondos Oscuros:** Blanco (`#FFFFFF`)
- **Acentos en Texto:** Verde (`#4A7C59`) o Naranja (`#D9773F`) para palabras clave

## Notas de Implementación

1. **Accesibilidad:** Todos los contrastes cumplen con WCAG AA mínimo
2. **Consistencia:** Mantener el azul oscuro como color de marca principal, reflejando el logo
3. **Jerarquía:** 
   - Azul oscuro para elementos principales y texto importante
   - Verde para elementos secundarios y acentos naturales
   - Naranja para acentos mecánicos/industriales
   - Gris azulado para texto secundario
4. **Espaciado:** Combinar con suficiente espacio en blanco para legibilidad
5. **Responsive:** Los colores deben mantenerse consistentes en todos los dispositivos
6. **Logo:** El logo debe mantenerse con sus colores originales (azul, verde, naranja, blanco) en todas las aplicaciones
7. **Temática:** Los colores reflejan la naturaleza (verde/montañas), industria (naranja/engranaje) y profesionalismo (azul oscuro)
8. **Tipografía:** Mantener consistencia con el estilo del logo - sans-serif, bold para títulos, uppercase para nombres de marca
